WatchCode‚Äã

∆¨ WatchCode: (event: "code", fn: (data: { code: Code ; config: Config }) => void) => { remove: () => void }

Called when the playground "content" is changed (see getCode and getConfig).

This includes changes in:

  • Code (in editors)
  • Editor languages
  • CSS processors
  • External resources
  • Project info (e.g. allows adding content in page head and attributes to <html> element)
  • Custom settings (e.g. allows changing import maps)
  • Project title
  • Test code
