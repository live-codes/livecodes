# cssnano

import LiveCodes from '../../src/components/LiveCodes.tsx';

[cssnano](https://cssnano.github.io/cssnano/) is a CSS minifier built on top of [PostCSS](./postcss.mdx). It applies various optimizations to reduce the size of your CSS output.

## Usage

cssnano is a [CSS processor](../features/css.mdx#css-processors) that can be enabled from the style editor menu.

When enabled, all CSS output (whether authored in CSS, [SCSS](./scss.mdx), [Sass](./sass.mdx), [Less](./less.mdx), [Stylus](./stylus.mdx), or any other style language) is minified and optimized.

cssnano can be used alongside other processors. Multiple processors are applied in the order they appear in the style editor menu.

### Optimizations

cssnano uses the [default preset](https://cssnano.github.io/cssnano/docs/presets/), which includes a wide range of optimizations such as:

- Removing whitespace and comments
- Minifying selectors and values
- Merging duplicate rules
- Reducing `calc()` expressions
- Normalizing values (e.g. colors, unicode ranges)

  border-radius: 8px;
  background-color: #ffffff;
  box-shadow: 0px 2px 8px rgba(0, 0, 0, 0.10);
}

  color: #2d3748;
  margin-top: 0px;
}

  color: #4a5568;
  line-height: 1.6;
}
`,
  },
  processors: ['cssnano'],
  tools: { active: 'compiled', status: 'open' },
  activeEditor: 'style',
};
